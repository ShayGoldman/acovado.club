## Overview

All infrastructure is self-hosted via Docker Compose. No cloud storage, no third-party auth provider, no payment integrations, no incoming/outgoing webhooks.

---

## Data Sources

### Reddit API
- **Type:** HTTP REST (unauthenticated)
- **Endpoints:** Public JSON endpoints at `www.reddit.com`
- **Client:** `modules/reddit-client/src/index.ts`
- **Pattern:** Decoupled via RabbitMQ request/response — the reddit-client module consumes fetch requests from a queue and publishes responses back
- **Limitations:** Anonymous rate limit (~60 req/min), no retry on 429

---

## Databases

### PostgreSQL 15
- **Type:** Relational database
- **Hosting:** Self-hosted via Docker
- **ORM:** Drizzle ORM
- **Client:** `modules/db/src/client.ts`
- **Connection:** `DATABASE_URL` env var
- **Schema:** 12 tables covering Reddit data (posts, comments, subreddits), watchlists, inference audit logs, and signal metrics
- **Notes:** Missing FK constraints on `stories` table; raw SQL used in some migrations

### FalkorDB (Graph Database)
- **Type:** Graph database (Redis protocol)
- **Hosting:** Self-hosted via Docker
- **Client:** `modules/graph-db/src/client.ts`
- **Connection:** `GRAPH_DB_URL` env var, accessed via `GRAPH.QUERY` commands over Redis protocol
- **Use case:** Graph relationships between entities (subreddits, posts, topics)
- **Notes:** Cypher query string interpolation used in some queries (injection risk)

---

## Messaging

### RabbitMQ
- **Type:** AMQP message broker
- **Hosting:** Self-hosted via Docker
- **Library:** `amqplib`
- **Producer/Consumer:** `modules/events/`
- **Connection:** `BROKER_URL` env var
- **Pattern:** Topic exchanges per domain
- **Notes:** No reconnection logic; `prefetch(1)` limits throughput; no dead-letter queue configured; credentials not scrubbed from logs

## AI / Inference

### Ollama (dev-only, removed from prod)
- **Type:** Local LLM server
- **Model:** `gemma3:12b-it-qat`
- **Client:** `modules/inference/src/client.ts` — model-agnostic wrapper with retry + DB audit logging
- **Status:** Removed from production Docker Compose (see commit history)
- **Notes:** Client remains in codebase; designed to be model-agnostic

## Observability

### SigNoz (self-hosted)
- **Version:** SigNoz v0.117.1
- **Components:** SigNoz + ClickHouse 25.12.9 + Zookeeper + OpenTelemetry Collector
- **Tracing SDK:** `modules/tracing/`
- **Export:** OTLP traces to `http://otel-collector:4318/v1/traces`
- **Access:** `signoz.acovado.club` (via Traefik)

## Infrastructure / Tooling

### Traefik
- **Type:** Reverse proxy / load balancer
- **Use case:** Routes management UIs, handles Let's Encrypt TLS termination
- **Domains:**
  - `rabbitmq.acovado.club` → RabbitMQ management UI
  - `signoz.acovado.club` → SigNoz observability UI
  - `stats.acovado.club` → Portainer

### Portainer CE
- **Type:** Docker management UI
- **Access:** `stats.acovado.club`
